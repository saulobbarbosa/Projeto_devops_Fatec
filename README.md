# DevStart

Protótipo de um site de **plataforma de cursos online** com foco em **tecnologia e DevOps**.

Este projeto foi desenvolvido como atividade da disciplina de **DevOps**, com o objetivo de praticar versionamento de código utilizando o **GitHub**.

## 📄 Sobre o projeto

O **DevStart** é um protótipo visual simples de uma plataforma de cursos online.
O site possui um design moderno com **tema dark**, utilizando **preto e cinza escuro** como cores principais e **vermelho** como cor de destaque.

O objetivo é apenas demonstrar a estrutura básica de um site, contendo três páginas principais.

## 🖥️ Telas do projeto

O protótipo possui 3 telas:

### 1. Home (Inicial)

* Apresentação da plataforma
* Título principal
* Botão de destaque
* Layout simples e organizado

### 2. Login

* Formulário de login
* Campos de email e senha
* Botão de entrar
* Layout centralizado

### 3. Contato

* Formulário de contato
* Campos de nome, email e mensagem
* Botão de envio

## 🎨 Estilo visual

* Tema **dark**
* Cores principais:

  * Preto / cinza escuro
  * Vermelho para destaque
  * Branco ou cinza claro para textos
* Design simples, moderno e minimalista
* Inspirado em sites de tecnologia

## 🛠️ Tecnologias utilizadas

* Versionamento com **Git** e **GitHub**

## 📚 Objetivo acadêmico

Este projeto foi desenvolvido para fins educacionais, como parte da disciplina de **DevOps**, com foco em:

* organização de projeto
* versionamento de arquivos
* uso do GitHub
* documentação com README
